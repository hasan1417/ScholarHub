Approach · System

One project graph. Three workflows. Shared context throughout.

A single shared substrate holds the library, discussions, and LaTeX. Every component reads through the same protocol — so the AI’s context is the project itself, not the open web.

01
Multi-Source Discovery
Nine academic databases, deduplicated into one project library.
02
Citation-Faithful Generation
AI cites only verifiable library identifiers — fabrication forbidden by protocol.
03
Real-Time Co-Editing
Multi-file LaTeX with live PDF preview, synchronised by Yjs CRDT.

Research Contribution

Three contributions of this thesis.

  1. i. Formalize the project-graph as a shared retrieval substrate spanning library, discussions, and editor.
  2. ii. Constrain AI generation to verifiable library identifiers, eliminating fabricated citations by construction.
  3. iii. Integrate real-time collaborative LaTeX editing with project-scoped AI in one workspace.
Future Work

Open problems and next steps.

  • Controlled user study with graduate researchers, comparing citation accuracy and writing throughput against baseline tools.
  • Open the project-graph protocol to third-party reference managers and writing assistants.
  • Domain-specialised retrieval tuning (medical, ML, materials science).
